Enrutamiento del router

Configuracion puerto de giga del router


en el primer interface normalmente se coloca el puerto con el que conecta la troncal,del otro dispositivo, en el segundo interface es la vlan con la que deseamos conectar, y en encapsulation solo tendremos que repetir el numero colocado en la interface, es decir si coloqué interface gi0/0.50 entonces tengo que poner "encapsulation dot1q 50" 

interface gi0/
no shutdown
interface gi0/0.
encapsulation dot1q 
ip address  
exit

Enrutamiento del switch 3560 o 3650

Configuracion puerto de giga del swicht


Se recomienda primero crear aqui las vlan necesarias y los nombres de esas vlan para que empiece a hacer el enrutamiento


Después de esto hay que hacerle troncal

interface vlan
ip address  
no shutdown
exit

troncal del switch 3560 o 3650 

interface Gi0/
switchport trunk encapsulation dot1q
switchport mode trunk
exit
ip routing
