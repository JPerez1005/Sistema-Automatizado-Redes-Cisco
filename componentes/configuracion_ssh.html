Recordar

A partir de aquí se supone que ya puede conectarse a este dispositivo desde una computadora por comand prompt.

password 
transport input ssh
login local
exit
enable secret  
do write

Entrar al swicht desde un pc

Pasos

se va a mostrar el codigo para conectarse desde un pc

ssh -l

Verificar configuración de ssh

Información

Aquí podremos ver algunas cuestiones, como ver si el SSH está activo, la versión que se está usando, tiempo maximo de autenticación, intentos en la autenticación, entre otras cosas con show run.

show ip ssh
show run
